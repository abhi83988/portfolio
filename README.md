# 🌐 Personal Portfolio Website

An interactive and visually rich personal portfolio website built using **React** and **Three.js**.  
It showcases my skills, projects, and experience with 3D elements, smooth animations, and a modern UI.

## ✨ Features

- Responsive design for mobile, tablet, and desktop.
- 3D scenes and effects powered by **Three.js / React Three Fiber**.
- Smooth animations and transitions with libraries like **Framer Motion** or **GSAP**.
- Dynamic sections for About, Skills, Projects, and Contact.
- Easy to customize and deploy.

## 🛠️ Tech Stack

- **React 18**
- **Three.js** (with React Three Fiber)
- **Framer Motion / GSAP** (animations)
- **Tailwind CSS / Styled Components** (styling)
- **EmailJS / Formspree** (contact form handling)
